# Profile Card (Stage 1B)

A responsive and accessible profile card built with HTML, CSS, and JavaScript.

## Features

- Semantic HTML structure
- Responsive design (mobile, tablet, desktop)
- Accessible UI (alt text, keyboard navigation)
- Displays current time in milliseconds using `Date.now()`
- Includes required `data-testid` attributes for testing
